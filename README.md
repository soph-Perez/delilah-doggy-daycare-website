# Delilah Doggy Daycare Website
A website that contains an about page, booking page, and contact page for the Delilah Doggy Daycare. The about page contains information about the company. The booking page allows users to book when and the type of day (full day or half day) that they are interested in coming in for. Lastly, the contact page allows users to send forms containing feedback

  ## How it's made:
  Developed with HTML, CSS, and Javascript
